# Auto English (자동 영타 전환기)

한/영 키 상태에 관계없이, 지정된 동작(엔터 키 입력, 한/영 키 커스텀 더블클릭, 유휴 시간 경과) 시 영문이나 한글로 상태를 자동 전환해 주는 Windows 전용 백그라운드 유틸리티입니다.

## 기능
- **엔터(Enter) 키 자동 전환**: 엔터 입력 시 영문/한글로 자동 전환
- **한/영 키 더블클릭 커스텀**: 단일 클릭과 더블클릭을 구분하여 각각 원하는 언어로 전환 (더블클릭 인식 제한 시간 밀리초 단위 설정 가능)
- **유휴 시간 자동 전환**: 설정한 시간(초) 동안 입력이 없으면 지정된 언어로 자동 전환
- **수동 오타 변환 트리거** (권장 순서):
  - **기호 연타** (`;;;` 등): 평소처럼 `;`가 입력됨. 드래그 선택 후에는 제대로 작동하지 않을 수 있음
  - **희귀키 연타** (Scroll Lock, `]`, `` ` ``, Caps Lock): 다른 프로그램과 충돌 적음
  - **마우스 가운데 클릭**: 드래그 변환에 유리 (일부 터미널에서는 붙여넣기로 동작할 수 있음)
  - 수식키 연타 / 조합 단축키: Shift 연타 시 Windows 고정 키 팝업에 주의 (설정 > 접근성 > 키보드 > 고정 키 끄기)
- **타겟 프로그램 감지**: VSCode, Chrome 등 등록한 특정 프로그램에서만 기능이 작동하도록 필터링
- **트레이 아이콘 실시간 연동**: 윈도우 우측 하단 트레이 아이콘에 백그라운드로 숨길 수 있으며, 아이콘 색상(파란색/회색)으로 현재 작동 상태(활성/비활성)를 실시간 확인 가능

## 요구 사항
- Windows OS (Windows API 직접 제어)
- Python 3.8 이상

## 설치 및 실행 방법
1. 이 레포지토리를 클론(Clone)하거나 다운로드합니다.
2. 터미널을 열고 필요한 패키지를 설치합니다:
   ```bash
   pip install keyboard pystray pillow
   ```
3. 프로그램을 실행합니다:
   ```bash
   python auto_eng_gui.pyw
   ```
   *(Windows 환경에서는 `auto_eng_gui.pyw` 파일을 더블 클릭하면 터미널 창 없이 백그라운드에서 깔끔하게 바로 실행됩니다.)*
